[SIZE="4"][color="#1E90FF"][B]CLEAN FLASH :[/B][/SIZE][/COLOR]
[SIZE="4"]Download the latest build and GApps (GApps only if vanilla)
Reboot to recovery 
Take Backup (Not Necessary)
Wipe System, Vendor, Data, Cache , Dalvik
Flash the latest build and GApps
Reboot[/SIZE]

[SIZE="4"][color="#1E90FF"][B]DIRTY FLASH :[/B][/SIZE][/COLOR]
[SIZE="4"]Download the latest build 
Reboot to recovery
Flash the latest build
Wipe Cache
Reboot[/SIZE]
Note : Don't flash GApps and Magisk again.
